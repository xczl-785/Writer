# 当前 Bug 统计

> 最后更新：2026-03-13

## 未闭环问题

1. 外部文件变更感知与刷新策略

- 现状：当文件在系统资源管理器或其他编辑器中被改动时，Writer 侧的刷新策略尚未完全闭环。
- 风险：可能出现编辑器内容与磁盘文件不一致。

2. 菜单项能力与可用态不完全一致

- 现状：顶部菜单已接入，但部分项仍为占位（STUB）或未按上下文禁用。
- 风险：用户会遇到"可见但不可用"的功能落差。

3. 多选文本统一样式操作

- 现状：跨片段多选后，字号/加粗/斜体等批量样式策略未形成统一规则。
- 风险：复杂选区下行为不稳定或不可预测。

4. Typewriter 模式下中文输入首字符阶段轻微抖动

- 现状：开启 Typewriter 后，新行中文输入时，前 1~2 个字符阶段仍可感知到轻微滚动回拉。
- 风险：会打断沉浸式输入节奏，影响"45% 锚点稳定"体验。

## 已解决问题（2026-03-13）

- ✅ Mac 平台粘贴触发 Paste 权限提示 → 已改为原生编辑菜单项 + 原生粘贴事件链，不再主动读取剪贴板
- ✅ 选中图片时无提示效果 → 已添加蓝色边框选中样式（`.ProseMirror img.ProseMirror-selectednode`）

## 1. 概述

### 1.1 文档目的

本文档描述 V6 工作区功能的交互设计规范，包括视觉设计、交互模式、动画规范和可访问性要求。

### 1.2 设计原则

| 原则   | 含义                 | 本功能应用                            |
| ------ | -------------------- | ------------------------------------- |
| 沉静   | 不炫技、不打扰       | 微妙的视觉反馈，避免炫目动画          |
| 专注   | 让内容成为焦点       | 工作区切换无声执行                    |
| 可信赖 | 行为可预测           | 一致的操作反馈，符合 VS Code 心智模型 |
| 无打断 | Non-Disruption First | 非模态通知优先                        |

## 2. 视觉设计

### 2.1 空状态界面

#### 2.1.1 布局结构

```
┌─────────────────────────────────────────┐
│                                         │
│           ✦ Writer ✦                   │  ← Logo/字标
│              写作，心流                 │  ← 标语
│                                         │
│    ┌────────────┐  ┌────────────┐       │
│    │  打开文件夹 │  │ 打开工作区 │       │  ← 主按钮
│    └────────────┘  └────────────┘       │
│                                         │
│    ───────── 最近打开 ─────────          │  ← 区域分隔线
│                                         │
│    💼 项目文档 (工作区)                 │  ← 工作区项
│    📁 ~/Documents/notes                 │  ← 文件夹项
│    📄 ~/writing/daily.md                │  ← 文件项
│                                         │
│    ────  或拖拽文件夹到此处  ────        │  ← 放置区提示
│                                         │
└─────────────────────────────────────────┘
```

#### 2.1.2 设计规范

| 元素     | 规范                                |
| -------- | ----------------------------------- |
| Logo     | 居中，距顶部 80px                   |
| 主按钮   | 并排居中，间距 16px，最小宽度 140px |
| 分隔线   | 1px solid zinc-200，上下留白 24px   |
| 最近项目 | 最多 10 项，垂直排列，间距 8px      |
| 图标     | 工作区💼、文件夹📁、文件📄，16x16   |
| 拖拽提示 | 斜体，zinc-400 色                   |

#### 2.1.3 状态说明

| 状态     | 描述                                           |
| -------- | ---------------------------------------------- |
| 默认     | 上述布局                                       |
| 拖拽悬停 | 整个区域边框 1px solid zinc-300 + 背景 zinc-50 |
| 拖拽无效 | 区域显示红色边框，提示"仅支持文件夹"           |

## 3. 交互模式

### 3.1 键盘快捷键

| 操作               | macOS           | Windows/Linux    | 备注                           |
| ------------------ | --------------- | ---------------- | ------------------------------ |
| 打开文件夹         | `Cmd+O`         | `Ctrl+O`         | 主入口                         |
| 关闭工作区         | `Cmd+Shift+W`   | `Ctrl+Shift+W`   | 与关闭文件区分                 |
| 关闭当前文件       | `Cmd+W`         | `Ctrl+W`         | 符合平台惯例                   |
| 添加文件夹到工作区 | `Cmd+Shift+A`   | `Ctrl+Shift+A`   | 简化快捷键                     |
| 保存工作区         | `Cmd+Shift+S`   | `Ctrl+Shift+S`   | 无标题工作区                   |
| 打开工作区文件     | `Cmd+Shift+O`   | `Ctrl+Shift+O`   | 打开已保存工作区               |
| 重新排序文件夹     | `Cmd+Shift+↑/↓` | `Ctrl+Shift+↑/↓` | 聚焦根文件夹时                 |
| 打开最近           | `Cmd+Shift+T`   | `Ctrl+Shift+T`   | 显示最近列表（避免与刷新冲突） |

### 3.2 右键菜单

### 3.3 拖拽操作

#### 3.3.1 外部拖拽（系统文件管理器 → Writer）

| 拖拽内容   | 放置区域       | 行为                                                 |
| ---------- | -------------- | ---------------------------------------------------- |
| `.md` 文件 | 编辑区         | 打开文件                                             |
| `.md` 文件 | 文件树         | 添加到工作区                                         |
| 文件夹     | 编辑区         | 打开为新工作区（提示）                               |
| 文件夹     | 文件树空白区域 | 添加到当前工作区（等同右键菜单"添加文件夹到工作区"） |
| 文件夹     | 文件树根节点上 | 添加到该根文件夹内                                   |

#### 3.3.2 内部拖拽（文件树内）

| 操作                   | 行为       | 视觉反馈        |
| ---------------------- | ---------- | --------------- |
| 拖拽文件到文件夹       | 移动文件   | 插入位置横线    |
| 拖拽文件夹到另一文件夹 | 移动文件夹 | 目标高亮        |
| 拖拽根文件夹 reorder   | 重新排序   | 插入位置横线    |
| `Cmd/Ctrl` + 拖拽      | 复制文件   | 光标显示 `+` 号 |

### 3.4 通知与反馈

#### 3.4.1 无标题工作区保存提示

**触发**: 关闭无标题工作区时

**形式**: 状态栏非模态通知

```
┌─────────────────────────────────────────────────┐
│ 💡 提示：多文件夹工作区可保存为 .writer-workspace 文件 │
│    [保存] [不保存] [不再提醒]                         │
└─────────────────────────────────────────────────┘
```

**行为**:

- 5 秒后自动消失
- 点击"不再提醒"后不再显示
- 点击"保存"打开保存对话框

#### 3.4.2 外部文件打开询问（非模态通知方案）

**触发**: 拖入工作区外文件时

**形式**: 状态栏非模态通知 + 可配置默认行为

**通知内容**:

```
┌─────────────────────────────────────────────────────────────┐
│ 📄 ~/other-project/notes.md 位于工作区外                    │
│    [打开] [复制到工作区] [设置默认行为]                      │
└─────────────────────────────────────────────────────────────┘
```

**行为**:

- 通知 5 秒后自动消失，默认执行"打开"操作
- 用户可在设置中配置默认行为：
  - 默认打开（推荐）
  - 默认复制到工作区
  - 始终询问（严格模式）
- 勾选"不再询问"后，按设置中的默认行为执行

**设置路径**: 设置 > 工作区 > 外部文件默认行为

## 4. 动画规范

### 4.1 过渡动画

| 场景            | 动画             | 时长  | 缓动            |
| --------------- | ---------------- | ----- | --------------- |
| 空状态淡入      | fade in          | 200ms | ease-out        |
| 文件树展开/折叠 | slide + fade     | 150ms | **ease-in-out** |
| 拖拽悬停高亮    | color transition | 100ms | linear          |
| 通知出现/消失   | slide + fade     | 200ms | ease-out        |

### 4.2 微交互

| 交互           | 反馈         | 时长  |
| -------------- | ------------ | ----- |
| 按钮 hover     | 背景色变化   | 100ms |
| 根文件夹 hover | 显示移除按钮 | 100ms |
| 保存成功       | 状态栏闪绿   | 500ms |
| 保存失败       | 状态栏闪红   | 500ms |

## 5. 可访问性

### 5.1 键盘导航

| 功能         | 键盘操作                 |
| ------------ | ------------------------ |
| 遍历根文件夹 | `Tab` / `Shift+Tab`      |
| 展开/折叠    | `Enter` 或 `Space`       |
| 重新排序     | `Cmd+Shift+↑/↓`          |
| 打开右键菜单 | `Shift+F10` 或 `Menu` 键 |
| 关闭工作区   | `Cmd+Shift+W`            |

### 5.2 屏幕阅读器

#### 5.2.1 ARIA 标签

```html
<!-- 根文件夹头部 -->
<div role="button" aria-label="根文件夹：项目文档" aria-expanded="true">
  📁 项目文档
</div>

<!-- 无标题工作区指示器 -->
<div role="status" aria-label="未命名工作区，未保存">未命名工作区 (未保存)</div>

<!-- 空状态界面 -->
<main role="main" aria-label="Writer 空状态">
  <!-- 内容 -->
</main>
```

#### 5.2.2 实时区域

```html
<!-- 工作区变更通知 -->
<div role="status" aria-live="polite" aria-atomic="true">
  已添加文件夹：用户手册
</div>
```

### 5.3 颜色对比度

| 元素     | 最小对比度 | 实际对比度                    | 状态 |
| -------- | ---------- | ----------------------------- | ---- |
| 普通文本 | 4.5:1      | ~15.3:1 (zinc-900 on white)   | ✅   |
| 次要文本 | 3:1        | 4.6:1 (zinc-500 on white)     | ✅   |
| 拖拽高亮 | 3:1        | **5.1:1 (blue-600 on white)** | ✅   |
| 按钮     | 3:1        | 4.5:1                         | ✅   |

**焦点指示器**:

- 颜色：`--color-focus-ring: #2563eb (blue-600)`
- 宽度：`--outline-width: 2px`
- 偏移：`--outline-offset: 2px`

### 5.4 运动减少

对于启用"减少运动"系统设置的用户：

| 动画     | 替代方案      |
| -------- | ------------- |
| 淡入淡出 | 直接显示/隐藏 |
| 滑入滑出 | 直接显示/隐藏 |
| 颜色过渡 | 立即切换      |

## 6. 边缘情况

### 6.1 文件夹不存在

**视觉**: 灰显，带警告图标

```
📁 已删除的项目 ⚠️ (不存在)
  [从工作区移除]
```

**交互**:

- 右键菜单仅显示"从工作区移除"
- 不可展开
- tooltip 显示完整路径和错误信息

### 6.2 工作区文件损坏

**视觉**: 状态栏错误通知

```
⚠️ 工作区文件损坏，无法加载
   [创建新工作区] [选择其他工作区]
```

### 6.3 拖拽冲突

| 场景                 | 处理                     |
| -------------------- | ------------------------ |
| 拖拽到无效区域       | 显示禁止图标，松开无操作 |
| 拖拽非 Markdown 文件 | 提示"不支持的文件类型"   |
| 拖拽时网络中断       | 提示"文件不可用"         |

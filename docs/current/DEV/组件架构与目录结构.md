## 二、组件详细设计

### 2.1 ContextMenu 组件

**职责**: 渲染右键菜单，支持动态菜单项注册

**Props**:

```typescript
interface ContextMenuProps {
  x: number; // 显示位置 X
  y: number; // 显示位置 Y
  context: string; // 上下文标识 (如 'filetree', 'editor')
  onClose: () => void;
}
```

**依赖**: `contextMenuRegistry.ts`

**样式规范**: `prototypes/01_context_menus.html`

- 背景: `bg-white dark:bg-zinc-800`
- 边框: `border border-zinc-200 dark:border-zinc-700`
- 阴影: `shadow-xl`
- 圆角: `rounded-md`
- 菜单项高度: `h-8` 或 `py-1.5 px-3`

### 2.2 BubbleMenu 组件

**职责**: 选中文本时显示格式化工具栏

**集成方式**: TipTap BubbleMenu 扩展

**样式规范**: `prototypes/01_context_menus.html`

- 背景: `bg-zinc-800 text-white`
- 边框: `border border-zinc-700`
- 阴影: `shadow-lg`
- 圆角: `rounded-md`

### 2.3 StatusIndicator 组件

**职责**: 显示文档保存状态 (四态流转)

**Props**:

```typescript
interface StatusIndicatorProps {
  status: 'saved' | 'dirty' | 'saving' | 'error';
  message?: string;
  errorDetails?: ErrorDetails;
}
```

**状态流转**:

```
saved ──(edit)──► dirty ──(save)──► saving ──(success)──► saved
  ▲                                              │
  │                                           (error)
  │                                              │
  └──────────────────────────────────────────────┘
                      │
                      ▼
                    error
```

**样式规范**: `prototypes/02_status_indicator.html`

- 已保存: `bg-emerald-500` 绿色圆点
- 未保存: `bg-amber-400` 黄色实心
- 保存中: `bg-blue-500 animate-pulse` 蓝色呼吸灯
- 错误: `bg-red-500` 红色 + 悬停面板

### 2.4 Outline 组件

**职责**: 显示文档大纲，支持点击跳转

**Props**:

```typescript
interface OutlineProps {
  editor: Editor;
  onItemClick?: (item: OutlineItem) => void;
}
```

**性能策略**:

- < 100 项: 全量渲染
- 100-500 项: 防抖更新 (150ms)
- > 500 项: 虚拟滚动

**样式规范**: `prototypes/03_outline_dialogs.html`

- 层级缩进: `pl-3` / `pl-6` / `pl-9`
- 激活项: `text-blue-600 bg-blue-50/50 border-blue-500`
- 悬停: `hover:text-blue-600 hover:bg-blue-50`

### 2.5 Dialog 组件

**职责**: 通用对话框容器 (用于非 Tauri 原生场景)

**Props**:

```typescript
interface DialogProps {
  open: boolean;
  onClose: () => void;
  title: string;
  children: React.ReactNode;
  actions?: React.ReactNode;
  size?: 'sm' | 'md' | 'lg';
}
```

**样式规范**: `prototypes/03_outline_dialogs.html`

- 遮罩: `bg-black/20`
- 面板: `bg-white border border-zinc-200 shadow-2xl rounded-lg`
- 宽度: `max-w-md`

## 三、状态管理设计

### 3.1 uiSlice (V5 新增)

```typescript
interface UIState {
  contextMenu: {
    open: boolean;
    x: number;
    y: number;
    context: string | null;
  };
  dialog: {
    open: boolean;
    type: 'delete-confirm' | 'error-detail' | null;
    data: Record<string, unknown>;
  };
  outline: {
    open: boolean;
    items: OutlineItem[];
    activeId: string | null;
  };
}
```

```typescript
interface StatusState {
  saveStatus: 'saved' | 'dirty' | 'saving' | 'error';

  errorDetails: ErrorDetails | null;
}
```

## 四、事件流设计

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户交互                                 │
├─────────────────────────────────────────────────────────────────┤
│  右键点击 ──► contextmenu event ──► useContextMenu.show()       │
│  文本选择 ──► selectionchange ──► BubbleMenu 自动显示            │
│  大纲点击 ──► onItemClick ──► editor.commands.setTextSelection  │
│  保存触发 ──► autosave.schedule ──► statusSlice.transition      │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                         状态更新                                 │
├─────────────────────────────────────────────────────────────────┤
│  uiSlice ──► contextMenu.open = true                            │
│  statusSlice ──► saveStatus = 'saving'                          │
│  editorSlice ──► outlineItems = extractedItems                  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                         视图渲染                                 │
├─────────────────────────────────────────────────────────────────┤
│  ContextMenu 组件渲染 ──► Portal 挂载到 body                     │
│  StatusIndicator 组件渲染 ──► 状态栏插槽                         │
│  Outline 组件渲染 ──► 侧边栏区域                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 六、文件命名规范

| 类型     | 命名规范             | 示例                       |
| -------- | -------------------- | -------------------------- |
| 组件     | PascalCase           | `ContextMenu.tsx`          |
| Hook     | camelCase + use 前缀 | `useContextMenu.ts`        |
| 工具函数 | camelCase            | `contextMenuRegistry.ts`   |
| 类型定义 | PascalCase           | `ContextMenuTypes.ts`      |
| 样式     | 同组件名             | `ContextMenu.css` (如需要) |
| 测试     | 同组件名 + .test     | `ContextMenu.test.ts`      |

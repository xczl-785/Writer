# V2 设计基线

## 1. 目标

确保 V2 新交互与 V1 体验一致且可学习成本可控。

## 2. 设计原则

1. 默认无打断写作。
2. 新能力优先键盘可达。
3. 失败反馈可见且可恢复。

## 3. 关键交互点

1. 表格插入、增删行列、单元格导航。
2. 搜索结果定位与焦点切换。
3. 顶部快捷菜单与快捷键行为一致性。

## 4. 顶部快捷菜单交互规范（MVP）

1. 一级按钮分组：样式组（B/I/Code）、块组（H1/H2/H3、列表、引用、代码块）、插入组（表格）。
2. 激活态规则：光标命中对应语义节点时按钮高亮。
3. 禁用态规则：编辑器未聚焦或上下文不可执行时按钮置灰。
4. 一致性规则：点击按钮与快捷键执行同一命令，不允许语义差异。

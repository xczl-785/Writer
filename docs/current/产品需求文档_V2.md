# 产品需求文档 V2（草案）

## 1. 目标

在保持 V1 写作体验稳定的前提下，补齐结构化编辑与信息组织能力。

## 2. V2 核心场景

1. 在文档中可视化编辑表格并正确保存。
2. 在工作区内快速定位目标文件。
3. 通过新建文件夹与导航提升组织效率。

## 3. 功能需求（V2 Must）

1. FR2-01：表格可视化编辑
2. FR2-02：应用内新建文件夹
3. FR2-03：快速文件搜索
4. FR2-04：查找/替换

## 4. 验收原则

1. 每个 FR 必须具备可执行 QA 用例。
2. 每个 FR 必须可关联证据路径。
3. 不得破坏 V1 Must 12 的回归通过状态。

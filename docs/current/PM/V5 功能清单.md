## 二、功能详情

### INT-001: 交互规范文档

**描述**: 定义该阶段所有新增层面的全局交互原子与法则准则库，保证迭代演进不形变。

**交付内容**:

- [x] 快捷键体系规范（含全局注册表与冲突策略定义）
- [x] 操作一致性原则（如：凡阻断必能撤销，凡高危必有确认）
- [x] 状态反馈规范（微动效标准时间 ≤300ms）
- [x] 交互术语表（统一 Bubble Menu / Context Menu 辞藻）

---

### INT-002: 右键菜单系统框架

**描述**: 管理系统各种不同范围触发的右键浮悬状态集合。

**状态**: ⏳ 待调研 (产品业务规则已录入，需转交前端架构师评估底层框架选型)

---

### INT-003: 状态反馈系统

**描述**: V5 不再容忍东一块西一块的临时 Toast，建立四种明确的文件保存流转状态，由视图底座同一处展示。

**四种状态设计方案**:

- [x] **已保存状态**: 常态隐匿，或保留极为轻量化的对勾小图标（绿色）。
- [x] **未保存状态**: 发生编辑 Dirty 标记后，状态变更为实心圆点（黄色）。
- [x] **保存中状态**: 触发自动/手动保存期间，变为呼吸灯效果或微型转圈（蓝色）。
- [x] **保存错误状态**: 保存遭遇系统级拦截（如断网、无权限），变为感叹号并可悬停展示错误体（红色）。

---

### INT-010: 文件树右键菜单

**描述**: 在左侧文件导航面板中，提供符合操作系统直觉的文件级上下文操作菜单。

**菜单项列表 (强制对齐 macOS Finder 体验)**: 
*   **新建文件** (Cmd+N)
*   **新建文件夹** (Shift+Cmd+N)
*   **---** (分隔线)
*   **重命名** (Enter)
*   **在访达中显示** (Reveal in Finder)
*   **复制路径** (Copy Path)
*   **---** (分隔线)
*   **移至废纸篓** (Cmd+Backspace, 红色高亮)

---

### INT-011: 编辑器右键菜单

**描述**: 根据编辑器内区域上下文，提供精准的行内操作辅助，而不是抛出一长串无关动作。

**场景区分规范**:

- [x] **选中文本时 (Bubble Menu 优先)**: 鼠标松开即悬浮显示。包含：**加粗**、*斜体*、<del>删除线</del>、`行内代码`、[超链接]、==高亮==。
- [x] **未选中时的系统右键 (Context Menu)**: 包含：粘贴、全选、复制全文、插入表格、插入引用。
- [x] **代码块内的右键**: 包含：复制代码块、变更语言、格式化代码。

---

### INT-012: 表格右键菜单

**描述**: 针对 Markdown 内部基于 TipTap 生成的表格区域，提供专属的快捷菜单（增删行列等）。

**状态**: ⏳ 待调研 (产品业务期望已确立，待开发介入评估 TipTap 的 Table 选区拦截逻辑)

---

### INT-013: 删除确认对话框

**描述**: V5 统一全系统的毁灭性动作的“二轮安保门票”。

**设计决策 (PO建议方案)**:

- [x] **实体文件删除 (左侧树)**: **必须**走原生系统对话框，利用 Tauri 调用 OS Native 弹窗询问“移至废纸篓？”。
- [x] **大段落内容删除 (编辑器内)**: **禁止弹窗**。直接删除。保障写作流畅。依靠无穷尽的 Undo 栈恢复即可。
- [x] **表格彻底删除 (编辑器内)**: **禁止弹窗**。直接删除。依赖 Undo。

---

### INT-014: 保存状态指示器

**描述**: 提供给用户定心丸的极简隐匿式指示。

**集成方式**:

- [x] **原生接入**: 挂载在主应用窗口底部最左/最右侧的状态栏插槽 (StatusBar Slot)。
- [x] **常态无扰**: 处于“已保存”稳态超过 5 秒后，指示器应当逐渐淡出降低不透明度（Opacity: 30%）。

---

### INT-015: 大纲导航面板

**描述**: 根据文档 Headers 自动提取并在侧边栏展示一键跳转长文的树状锚点。

**状态**: ⏳ 待开发 (体验规范要求见《验收文档》，核心技术攻坚项已剥离至架构评估报告)

---

### INT-016: 错误处理规范化

**描述**: 将系统级的不可继续操作（读写无权限等）捕获，并转化给用户友好的、包含具体“动作建议”的 UI。

**状态**: ⏳ 待开发 (底层如何与老版 ErrorService 集成由前端研发处理)



---

### INT-017: 块级元素边界提示

**描述**: 当光标进入代码块、引用块等块级元素时，通过左侧装订线 提供清晰的视觉边界，帮助用户理解文档结构。

**对应需求**: US-B3 (Must 级别)

**交付内容**:

- [x] 代码块装订线: 左侧 4px 宽度蓝色高亮线
- [x] 引用块装订线: 左侧 2px 宽度灰色引用线
- [x] 光标进入/离开动画: 50ms 显示, 100ms 隐藏

**技术实现**: TipTap Decoration + ProseMirror Plugin

---

### INT-018: 面包屑路径导航

**描述**: 在编辑器顶部显示当前文件的路径面包屑，帮助用户快速理解文件在工作区中的位置，并支持点击跳转。

**对应需求**: US-A2 (Should 级别)

**交付内容**:

- [x] 路径层级显示: 支持点击跳转对应目录
- [x] 长路径截断: 超过 4 级时中部截断，保留首项和末两项
- [x] 悬停提示: 截断区域显示完整路径 Tooltip

**技术实现**: Breadcrumb 组件 + workspaceSlice 状态读取

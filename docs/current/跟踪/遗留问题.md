# 遗留问题

> 用于记录后续待推进的需求项（非 Bug）。

## 当前遗留

1. 文档加载性能待优化

- 现状：当前打开文档时，对比其他同类软件，加载速度稍慢
- 缺口：性能优化未专项推进
- 建议后续：专项调研瓶颈点并优化

2. 结构性感知待优化

- 现状：参考 ./design-block-indicator-system.md
- 缺口：设计存在4个阻塞项未解决
- 建议后续：先补齐阻塞项再进入开发

3. Slash Menu中，部分功能缺失，只有入口

- 现状：菜单项存在但功能未实现
- 缺口：产生用户困惑
- 建议后续：补齐功能或隐藏入口

4. 拖拽场景边界待明确

- 现状：当前拖拽功能已实现文件/文件夹区分显示不同提示，但多选拖入场景行为待确认。
- 缺口：
  - 混合拖入（文件+文件夹）：当前优先处理文件，忽略文件夹，需确认期望行为（阻止/分别处理/其他）。
  - 多文件拖入：当前只处理第一个文件，需确认是否支持批量打开（多标签页）。
- 建议后续：明确产品期望后，补充相应的交互设计和实现。

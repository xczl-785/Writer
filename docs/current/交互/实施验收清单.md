## 3. 已知差异与后续

- Focus Zen 独立快捷键（如 F11）尚未新增绑定。本轮保留双击侧栏按钮与 Esc 退出链路，待键位决策后补齐命令映射。
- 持久化当前为 `Zustand persist + web storage adapter`，后续需按迁移策略平滑切换到 Tauri store。

## 4. 模块 C（Focus Zen）遗留项

- Esc 退出的“活跃浮层”当前覆盖 `Slash Menu / Find Panel / Context Menu / Outline`；若后续新增编辑器浮层类型，需要同步纳入 Esc 优先级守卫。
- Focus Zen 的专用快捷键尚未固化（当前仍以双击侧栏按钮为主入口）；待键位方案确认后补齐并回归 `TC-C01~TC-C09`。
- 双击交互已改为“单击延时 + 双击优先”，需持续人工回归高频连击场景（`TC-C09`）以监控不同平台下的双击阈值体验差异。

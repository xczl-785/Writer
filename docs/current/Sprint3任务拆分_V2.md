# Sprint3 任务拆分（V2 执行）

## 目标

完成交互细节完善与失败恢复能力建设，确保搜索/查找替换在复杂场景下稳定可用。

## 任务列表

- [ ] S3-V2-01 搜索体验优化（空结果态、键盘导航、高亮一致性）
- [ ] S3-V2-02 查找/替换增强（全部替换、结果计数、边界提示）
- [ ] S3-V2-03 失败恢复链路（异常提示、可重试、状态回收）
- [ ] S3-V2-04 性能与稳定性优化（大文件场景响应、输入卡顿治理）
- [ ] S3-V2-05 跨功能回归门禁（表格、顶部菜单、搜索、替换联动）

## DoD

1. 搜索/查找替换交互在常见边界场景下行为清晰。
2. 出现失败时有明确提示且支持恢复操作。
3. 大文件场景下关键交互无明显阻塞。
4. 关键链路回归通过，未引入 P0/P1 级问题。

# QA 验收清单 Sprint 1（编辑核心闭环）

## 1. 目的

本清单用于 Sprint 1 的手工回归验证，确保“最小可写作闭环”稳定可用，并且不出现静默数据丢失。

## 2. 范围

- Sprint 1 功能：启动、文件树、编辑器、自动保存、关闭前保护。
- 核心路径：打开目录 -> 打开文件 -> 编辑 -> 自动保存 -> 关闭重开。
- 质量目标：无静默失败、错误可见、回归可复现。

## 3. 环境基线

- 平台：macOS（主验收）/ Windows（补充验证）。
- 工作区：单根目录。
- 样例文件：
  - `sample-plain.md`（普通段落）
  - `sample-list.md`（有序/无序列表）
  - `sample-code.md`（代码块）

# Sprint 2.5 交互修正回归用例

## 执行说明
- 平台：先 macOS，再 Windows。
- 每条用例填写：`PASS / FAIL / BLOCKED`。
- 建议同时录屏，方便复盘焦点与 blur 行为。

## A. 重命名交互

| 用例ID | 步骤 | 预期 | 结果 | 备注 |
| --- | --- | --- | --- | --- |
| S25-RN-01 | 选择文件，点重命名按钮 | 进入行内输入态；无 Save/Cancel 按钮 |  |  |
| S25-RN-02 | 输入新名称按 Enter | 重命名成功并退出输入态 |  |  |
| S25-RN-03 | 输入新名称按 Esc | 取消重命名并恢复旧名称 |  |  |
| S25-RN-04 | 输入非法名称按 Enter | 弹错误提示；回滚到旧名称并退出 |  |  |
| S25-RN-05 | 输入合法名称，鼠标点外部 | 视为提交；重命名成功 |  |  |
| S25-RN-06 | 输入非法名称，鼠标点外部 | 静默回滚旧名称并退出 |  |  |
| S25-RN-07 | 重命名输入态时观察行高 | 行高不变，树结构不抖动 |  |  |

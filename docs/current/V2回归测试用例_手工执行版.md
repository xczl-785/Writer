# V2 回归测试用例 (手工执行版)

## 1. 执行说明与环境准备

### 1.1 执行环境

- **推荐模式**：Tauri Native 桌面客户端 (Windows/macOS/Linux)。
- **禁用模式**：浏览器 Web 模式 (http://127.0.0.1:4173/)。
- **原因**：Web 模式下原生文件夹选择器 (Native File Picker) 存在已知阻塞，无法完成端到端验证。

### 1.2 前置准备

1. 确保应用已通过 `npm run tauri dev` 或安装包启动。
2. 准备一个包含至少 5 个 Markdown 文件和 2 个子文件夹的测试目录。
3. 确保测试目录下包含表格、代码块等 V2 新增元素。

## 一、项目概览

### 1.1 V5 目标

建立统一的交互风格与规范，为未来扩展打下坚实基础。

### 1.2 技术栈

- Tauri 2 + Rust (后端)
- React 19 + TypeScript (前端)
- TipTap (ProseMirror) (编辑器)
- Zustand (状态管理)
- Tailwind CSS 4 (样式)

### 1.3 设计原则

- **无打断** - 任何交互不应打断用户写作心流
- **一致性** - 相同操作在所有场景下表现一致
- **可发现** - 功能应有学习路径
- **渐进式** - 从简单到复杂逐步学习

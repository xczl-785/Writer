# Scope Freeze V1（冻结版）

## 冻结声明

本文件用于冻结 Writer V1 范围。自本文件确认后：

- V1 只实现 Must 12。
- Should/Could 统一进入 Backlog。
- Out 项目不进入 V1。

## V1 Must（12 项，冻结）

### A. 编辑核心（7）

1. 即时渲染编辑（所见即所得）
2. 段落级格式快捷键（标题/列表/引用）
3. 行内样式快捷键（加粗/斜体/行内代码）
4. 代码块编辑（含基础语法高亮）
5. 图片粘贴与本地保存（assets 相对路径）
6. 自动保存
7. 撤销/重做（多步）

### B. 文件管理（5）

8. 侧边栏文件树
9. 应用内新建文件
10. 文件重命名（文件树与已打开标签状态一致）
11. 文件删除（含确认）
12. 文件夹作为工作区（打开目录即开始）

## 变更控制规则

1. 任何新增功能默认进入 Backlog，不直接进入 Must。
2. 仅当功能满足以下条件，才允许替换 Must：
   - 直接服务无打断写作主流程
   - 可在一次回归中稳定验收
   - 不显著拖慢启动速度与交互响应
3. Must 变更需要单独记录“替换原因 + 影响范围 + 验收更新”。

# Writer V5 窗口与交互尺寸清单

本文档汇总了 Writer V5 设计方案中涉及的所有核心尺寸参数，供研发实现与视觉核对使用。

## 1. 窗口尺寸标准 (Window Sizes)

| 场景 | 宽度 (px) | 高度 (px) | 侧边栏行为 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **最小限制 (Min)** | 640 | 480 | 初始收起 (Collapsed) | 禅模式触发临界点 |
| **标准 HD (720p)** | 1280 | 720 | 默认开启 (240px) | 主流笔记本外显兼容区 |
| **推荐默认尺寸** | 1180 | 800 | 默认开启 (240px) | MacBook 14" 全屏感官最优 |
| **高分屏区 (Hi-DPI)** | 1440+ | 900+ | 保持开启 | 锁定正文列宽，增加两侧留白 |

## 2. 编辑器排版参数 (Editor Spacing)

| 参数项 | 模式 | 推荐值 | 说明 |
| :--- | :--- | :--- | :--- |
| **正文字号** | 全模式 | 16px | 物理字号基准，全档位恒定 |
| **正文列宽** | Airy / Default | 850px | 自适应居中，最大不超过 850px |
| **顶部留白** | Default / Airy | **64px** | 保证视觉呼吸感 |
| **顶部留白** | **Min (Zen)** | **32px** | 空间受限，优先显示文字 |
| **左右内边距** | Default / Airy | **32px** | 黄金边距 |
| **左右内边距** | **Min (Zen)** | **16px** | 缩减边距以增加单行字符数 |
| **底部留白** | 全模式 | 40vh | 保证光标滚动余地 |

## 3. 悬浮组件参数 (Floating UI)

### Slash Menu (斜杠菜单)
- **单项高度**: 34px
- **圆角**: 12px
- **水平行为**: Right-Edge Aware (自动向左偏移以防出界)
- **空间保护**: 若视口高度 < 500px，开启内滚，**Max-Height: 85%**

### 设置弹窗 (Settings Dialog)
- **标准规格**: 640px x 480px, Modal (带遮罩)
- **Min 模式下规格**: 自动占满（四周留白 **Inset: 20px**）, 取消大阴影

## 4. 极致禅模式参数 (Focus Zen)

| 参数项 | 推荐值 | 说明 |
| :--- | :--- | :--- |
| **UI 消隐透明度** | 0% - 10% | 极致禅开启后，非编辑区 UI 的初始透明度 |
| **悬浮唤醒区** | 50px | 靠近屏幕顶/底 edge 的激活高度 |
| **渐变时长** | 200ms | UI 浮现/消失的 transition 时间 |

## 5. 线条与精致度 (Retina Refinement)

| 环境 | 线条粗细 | CSS 参考 |
| :--- | :--- | :--- |
| **普通显示器 (1x)** | 1px | `border: 1px solid #E4E4E7` |
| **Retina/Hi-DPI (2x+)** | 0.5px | `border: 0.5px solid #E4E4E7` 或使用 `scale` 模拟极细线 |

## 6. 实现映射备注（2026-03-04）

- 窗口默认/最小尺寸已落地：`1180x800` 与 `640x480`。
- 响应式档位已落地：`<=640 / 641-1439 / >=1440`，并附 `200ms` 防抖。
- Editor 间距已落地：Top `64/32`、Side `32/16`、Max Width `850`、Bottom `40vh`。
- Focus Zen 唤醒区与过渡已落地：Top/Bottom `50px`，`opacity 0.2s`。
- Retina hairline 已通过全局变量 `--hairline-width` 实现（1x=1px，2x+=0.5px）。
- Slash Menu 小窗保护实现为 `max-height: 85vh`（语义等价于 85% 视口高度）。

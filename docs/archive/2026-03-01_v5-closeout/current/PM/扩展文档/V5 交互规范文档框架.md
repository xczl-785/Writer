## 1. 设计原则

### 1.1 无打断原则 (Non-Disruption)

**定义**: 任何交互不应打断用户的写作心流。

**规范细则**:

| 场景         | 正确做法                 | 错误做法            |
| ------------ | ------------------------ | ------------------- |
| **自动保存** | 状态栏无声更新指示器     | 保存成功 Toast 弹窗 |
| **错误处理** | 状态栏显示错误，继续编辑 | 模态对话框阻断编辑  |
| **系统通知** | 角标提示，用户主动查看   | 弹窗打断            |
| **功能引导** | 非模态温和提示           | 强制教程            |

**量化指标**:

- 编辑区弹窗数量 = 0 (除二次确认外)
- 非紧急通知延迟显示 (用户停止输入 3 秒后)
- 自动保存无感知 (延迟 < 100ms)

---

### 1.2 一致性原则 (Consistency)

**定义**: 相同操作在所有场景下表现一致。

**规范细则**:

| 操作类型       | 一致行为规范                    |
| -------------- | ------------------------------- |
| **破坏性操作** | 统一使用模态对话框二次确认      |
| **右键菜单**   | 统一在点击位置显示，ESC 关闭    |
| **快捷键**     | Mod 键统一为 Cmd(Mac)/Ctrl(Win) |
| **状态反馈**   | 统一在状态栏左侧显示            |
| **错误提示**   | 统一格式：错误原因 + 建议操作   |

**违反示例**:

- ❌ 文件删除有确认，文本删除无确认
- ❌ 某些菜单点击关闭，某些菜单需 ESC 关闭
- ❌ 状态指示器位置不固定

---

### 1.3 可发现原则 (Discoverability)

**定义**: 功能应有学习路径，不完全隐藏。

**规范细则**:

| 功能类型     | 发现路径                      |
| ------------ | ----------------------------- |
| **基础功能** | 工具栏/右键菜单可见           |
| **进阶功能** | 右键菜单 + 快捷键提示         |
| **高级功能** | 设置面板 + 快捷键速查 (Mod+?) |

**实现要求**:

- 所有右键菜单项支持键盘导航 (↑↓选择，Enter 确认)
- 悬停菜单项 >500ms 显示快捷键提示
- 提供快捷键速查面板 (Mod+?)

---

### 1.4 渐进式原则 (Progressive)

**定义**: 交互密度允许用户从简单到复杂逐步学习。

**规范细则**:

| 用户阶段              | 可接触功能          | 界面复杂度        |
| --------------------- | ------------------- | ----------------- |
| **新手** (首次启动)   | 新建/打开/编辑/保存 | 仅工具栏 + 编辑区 |
| **熟练** (使用 1 周+) | 右键菜单/快捷键     | 侧边栏 + 状态栏   |
| **专家** (主动探索)   | 自定义设置/高级功能 | 设置面板全量开放  |

**实现要求**:

- 首次启动不显示功能引导弹窗
- 设置面板分"基础/高级"两档
- 高级功能默认隐藏，可手动开启

## 2. 交互模式

### 2.1 编辑模式

**定义**: 用户主要内容创作状态。

**进入条件**: 打开文件或新建文件。

**交互特征**:

- 工具栏可折叠 (默认展开)
- 侧边栏可折叠 (默认展开)
- 状态栏常驻显示
- 所有编辑操作响应延迟 < 50ms

**退出条件**: 关闭文件或切换文件。

---

### 2.2 导航模式

**定义**: 用户在文件树中浏览/管理文件状态。

**进入条件**: 点击侧边栏文件树区域。

**交互特征**:

- 文件树获得焦点 (视觉高亮)
- 键盘导航激活 (↑↓选择，Enter 打开)
- 右键菜单可用

**退出条件**: 点击编辑区或关闭侧边栏。

---

### 2.3 搜索模式

**定义**: 用户使用搜索功能状态。

**进入条件**: 触发搜索命令 (Mod+P 文件搜索 / Mod+F 内容搜索)。

**交互特征**:

- 搜索面板滑出
- 焦点自动进入搜索框
- ESC 关闭搜索面板

**退出条件**: ESC 关闭或选择搜索结果后自动关闭。

---

### 2.4 命令模式

**定义**: 用户使用命令面板执行操作状态。

**进入条件**: 触发命令面板 (Mod+Shift+P)。

**交互特征**:

- 命令面板居中显示
- 支持模糊搜索命令
- ↑↓选择，Enter 执行，ESC 关闭

**退出条件**: 执行命令后或 ESC 关闭。

## 3. 组件规范

### 3.1 右键上下文菜单

**使用场景**:

- 文件树区域右键
- 编辑器区域右键

**显示规则**:

- 位置：鼠标点击位置右下偏移 (8px, 8px)
- 边界：不超出窗口边界，自动调整
- 层级：z-index 最高 (覆盖所有内容)

**交互规则**:

- 点击菜单项：执行命令 → 关闭菜单
- 点击菜单外：直接关闭菜单
- ESC 键：关闭菜单
- 键盘导航：↑↓选择，Enter 确认

**菜单项分组**:

- 使用分隔线分隔逻辑组
- 不使用嵌套子菜单 (Writer 场景保持扁平)
- 每组不超过 5 项

**性能要求**:

- 显示延迟 < 50ms
- 菜单项数量 ≤ 12 项

---

### 3.2 对话框与确认

**使用场景**:

- 删除操作二次确认
- 重要设置变更确认

**类型定义**:

| 类型           | 使用场景       | 行为               |
| -------------- | -------------- | ------------------ |
| **模态对话框** | 破坏性操作确认 | 阻塞编辑，必须响应 |
| **非模态提示** | 信息提示       | 不阻塞，自动消失   |

**模态对话框规范**:

- 背景遮罩 (透明度 50%)
- 居中显示
- 焦点锁定 (Tab 键循环)
- ESC 可关闭 (除非明确禁止)

**按钮规范**:

- 确认按钮：右侧，红色 (破坏性) / 蓝色 (普通)
- 取消按钮：右侧，灰色
- 按钮文字：明确动作 (删除/取消)

---

### 3.3 状态指示器

**使用场景**: 状态栏显示各类状态信息。

**保存状态指示器**:

| 状态         | 视觉表现                       | 颜色 |
| ------------ | ------------------------------ | ---- |
| **已保存**   | ✓ 图标 + "已保存"              | 绿色 |
| **未保存**   | ● 图标 + "未保存"              | 灰色 |
| **保存中**   | ⏳ 图标 + "保存中..."          | 蓝色 |
| **保存错误** | ⚠ 图标 + "保存失败" + 错误原因 | 红色 |

**位置**: 状态栏左侧。

**更新规则**:

- 状态变化立即更新 (< 100ms)
- 无动画过渡 (避免干扰)

---

### 3.4 工具栏

**使用场景**: 编辑区顶部，提供常用格式操作入口。

**交互规则**:

- 支持折叠/展开
- 按钮悬停显示 Tooltip (含快捷键)
- 选中文字时自动显示相关工具

**布局规范**:

- 左侧：格式操作 (粗体/斜体/标题)
- 中间：插入操作 (链接/图片/表格)
- 右侧：视图操作 (侧边栏切换/专注模式)

---

### 3.5 侧边栏

**使用场景**: 文件树/大纲导航。

**交互规则**:

- 支持拖拽调整宽度
- 支持折叠/展开 (Mod+B)
- 文件树获得焦点时键盘导航激活

**最小宽度**: 200px。

**最大宽度**: 窗口宽度的 50%。

## 4. 反馈规范

### 4.1 状态反馈

**原则**: 被动展示，不打断。

**规范**:

| 状态类型     | 反馈位置   | 反馈形式    | 持续时间   |
| ------------ | ---------- | ----------- | ---------- |
| **保存状态** | 状态栏左侧 | 图标 + 文字 | 常驻       |
| **光标位置** | 状态栏右侧 | 行列号      | 常驻       |
| **选中字数** | 状态栏右侧 | 数字        | 选中时显示 |
| **文件大小** | 状态栏右侧 | 数字        | 常驻       |

---

### 4.2 错误处理

**原则**: 清晰可理解，包含原因 + 建议。

**错误信息格式**:

```
[错误类型] 错误原因。建议操作。
```

**示例**:

- ✅ "保存失败：磁盘空间不足。请清理磁盘后重试。"
- ✅ "重命名失败：文件已存在。请使用其他文件名。"
- ❌ "错误：操作失败" (无原因无建议)

**错误分级**:

| 级别           | 反馈形式              | 是否阻断 |
| -------------- | --------------------- | -------- |
| **致命错误**   | 模态对话框            | 是       |
| **可恢复错误** | 状态栏红色提示 + 建议 | 否       |
| **警告**       | 状态栏黄色提示        | 否       |

---

### 4.3 加载反馈

**使用场景**: 耗时操作 (>500ms)。

**规范**:

| 耗时范围   | 反馈形式            |
| ---------- | ------------------- |
| < 500ms    | 无反馈 (无感知)     |
| 500ms - 2s | 状态栏加载指示器    |
| > 2s       | 进度对话框 (可取消) |

**加载指示器**:

- 位置：状态栏左侧 (临时占用保存状态位置)
- 形式：旋转图标 + 文字说明
- 可取消操作：显示"取消"按钮

---

### 4.4 成功反馈

**原则**: 非紧急操作无反馈，紧急操作简短反馈。

**规范**:

| 操作类型         | 反馈形式                                   |
| ---------------- | ------------------------------------------ |
| **保存成功**     | 无反馈 (状态栏更新即可)                    |
| **文件创建成功** | 无反馈 (直接显示文件)                      |
| **复制成功**     | 状态栏简短提示 "已复制" (1 秒消失)         |
| **导入成功**     | 状态栏提示 "导入完成，N 个文件" (2 秒消失) |

**禁止**:

- ❌ Toast 弹窗
- ❌ 成功提示对话框

## 5. 快捷键规范

### 5.1 快捷键体系

**修饰键定义**:

| 键位          | Mac       | Windows/Linux |
| ------------- | --------- | ------------- |
| **Mod**       | Cmd       | Ctrl          |
| **Mod+Shift** | Cmd+Shift | Ctrl+Shift    |
| **Alt**       | Option    | Alt           |

**快捷键分类**:

| 类别       | 修饰键             | 示例               |
| ---------- | ------------------ | ------------------ |
| **编辑类** | Mod + 字母         | Mod+B (粗体)       |
| **导航类** | Mod + 数字         | Mod+1 (H1)         |
| **系统类** | Mod + Shift + 字母 | Mod+Shift+S (设置) |
| **全局类** | 无修饰             | F5 (刷新)          |

---

### 5.2 冲突处理

**优先级规则**:

1. 系统快捷键 > 应用快捷键
2. 编辑区快捷键 > 全局快捷键
3. 新定义快捷键不得与现有快捷键冲突

**冲突检测**:

- 每次新增快捷键前执行冲突检测
- 发现冲突时报错并阻止注册

### 5.3 可发现性

**快捷键提示**:

| 场景           | 提示方式                |
| -------------- | ----------------------- |
| **工具栏按钮** | 悬停 Tooltip 显示快捷键 |
| **右键菜单项** | 右侧显示快捷键          |
| **命令面板**   | 命令右侧显示快捷键      |
| **快捷键速查** | Mod+? 打开速查面板      |

**速查面板规范**:

- 分类展示 (编辑/导航/系统)
- 支持搜索快捷键
- 支持修改快捷键 (V6)

## Sprint 1（编辑核心闭环）

### S1-06c 斜体快捷键无效（非阻断）

- 问题现象：`Cmd+I` 无反应，加粗 `Cmd+B` 可用。
- 影响范围：编辑器基础格式化体验不一致。
- 修复方案：
  - 在编辑器中增加快捷键扩展，显式绑定：
    - `Mod-i -> toggleItalic`
    - `Mod-b -> toggleBold`
- 修改文件：
  - `src/ui/editor/Editor.tsx`
- 验证结果：通过（手动验证 + 构建通过）。

### S1-06d 代码块视觉对比度不足（非阻断）

- 问题现象：代码块背景与周围区域对比度偏低，辨识不清。
- 影响范围：阅读体验和可读性。
- 修复方案：
  - 提升 `pre/code` 区域对比度、边框和内边距；
  - 调整行内代码样式，使其与普通文本区分更明显。
- 修改文件：
  - `src/ui/editor/Editor.css`
- 验证结果：通过（样式回归 + 构建通过）。

### S1-08 打开目录响应迟缓（非阻断）

- 问题现象：点击 `Open Folder` 后响应较慢。
- 影响范围：目录选择启动速度。
- 修复方案：
  - 目录选择对话框去除 `recursive: true`，避免不必要递归行为。
- 修改文件：
  - `src/workspace/WorkspaceManager.ts`
- 验证结果：通过（功能可用 + 构建通过）。

## Sprint 2（文件管理闭环）

### S2-03c 子目录新建文件落到根目录（阻断）

- 问题现象：选中子目录后新建文件，文件仍创建在工作区根目录。
- 影响范围：新建文件路径错误。
- 修复方案：
  - 引入“当前选中节点”路径优先策略；
  - 选中目录：在目录下新建；选中文件：在同级目录新建；否则回退到 activeFile/根目录。
- 修改文件：
  - `src/ui/sidebar/Sidebar.tsx`
  - `src/ui/sidebar/pathing.ts`
  - `src/ui/sidebar/pathing.test.ts`
- 验证结果：通过（测试通过 + 构建通过）。

### S2-04 重命名交互异常（阻断）

- 问题现象：重命名入口不可用或体验不稳定。
- 影响范围：文件/目录重命名流程。
- 修复方案：
  - 改为树节点内联重命名输入态（`Save/Cancel` + `Enter/Escape`）；
  - 保留重命名前 flush 安全链路。
- 修改文件：
  - `src/ui/sidebar/Sidebar.tsx`
- 验证结果：通过（手工验证 + 构建通过）。

### S2-05 删除确认时序异常（阻断）

- 问题现象：确认弹窗仍在时节点已被删除，取消后无法还原。
- 影响范围：删除安全性和用户预期。
- 修复方案：
  - 删除改为独立确认弹窗；
  - 仅在用户点击确认后执行删除；
  - 增加“Do not ask again（coming soon）”置灰入口，预留后续配置能力。
- 修改文件：
  - `src/ui/sidebar/Sidebar.tsx`
- 验证结果：通过（手工验证 + 构建通过）。

### S2 文件树显示策略优化（体验项）

- 需求约束：当前产品仅展示 Markdown 文件。
- 实施内容：
  - 仅显示 `.md/.markdown/.mdx` 文件；
  - 过滤非 Markdown 文件；
  - 空目录在过滤后不展示；
  - 文件名展示隐藏后缀；
  - 新建文件默认补 `.md`；
  - 重命名文件时输入不带后缀，提交后自动保留原后缀。
- 修改文件：
  - `src/ui/sidebar/pathing.ts`
  - `src/ui/sidebar/pathing.test.ts`
  - `src/ui/sidebar/Sidebar.tsx`
- 验证结果：通过（测试通过 + 构建通过）。

## 2. 架构决策

### 2.1 核心架构决策

| 决策                 | 方案                                   | 理由                                   |
| -------------------- | -------------------------------------- | -------------------------------------- |
| 状态模型             | 单一 `folders[]` 数组                  | 避免 `currentPath` 和 `folders` 冗余   |
| 工作区类型           | 计算属性，不存储                       | 从 `folders.length` 推导               |
| 文件树结构           | 多根节点 `RootFolderNode[]`            | 支持多个独立根文件夹                   |
| 事务协调             | `workspaceActions` + 快照回滚          | 避免多 store 竞态条件，支持失败恢复    |
| 后端命令             | 批量操作优先                           | 减少 IPC 往返次数                      |
| 文件监听             | **Rust notify crate + Tauri 事件推送** | Tauri 前端无法使用 Node.js 模块        |
| 路径安全             | WorkspaceAllowlist 验证层              | 防止路径穿越攻击                       |
| **Watcher 生命周期** | **全局状态管理（Arc<Mutex<Option>>）** | **避免 watcher 被 drop，支持停止监听** |

### 2.2 架构变更摘要

| 模块                       | 变更类型 | 变更描述                             |
| -------------------------- | -------- | ------------------------------------ |
| `workspaceSlice`           | 重构     | 移除 `currentPath`，使用 `folders[]` |
| `filetreeSlice`            | 扩展     | 添加 `rootFolders[]` 支持多根        |
| `workspaceActions`         | 新建     | 多 store 事务协调器（带快照 - 回滚） |
| `FsService`                | 扩展     | 添加批量命令和工作区命令             |
| `WorkspaceMetadataService` | 新建     | 工作区元数据管理                     |
| `FileWatcherService`       | **重写** | **改为后端监听 + 前端订阅模式**      |
| `WorkspaceSecurity`        | 新建     | 路径安全验证层                       |

### 3.3 WorkspaceActions 协调器（带快照 - 回滚机制）

```typescript
// src/state/actions/workspaceActions.ts

interface WorkspaceSnapshot {
  folders: WorkspaceFolder[];
  rootFolders: RootFolderNode[];
}

export const workspaceActions = {
  // 现有方法
  openFile: async (path: string): Promise<OpenFileResult> => { ... },
  loadWorkspace: async (path: string): Promise<number> => { ... },
  closeWorkspace: () => { ... },

  // 新增方法 - 多文件夹支持（带快照回滚）
  addFolderToWorkspace: async (folderPath: string): Promise<Result> => {
    // 1. 检查是否已存在
    const currentFolders = useWorkspaceStore.getState().folders;
    if (currentFolders.some(f => f.path === folderPath)) {
      return { ok: false, error: 'Folder already exists' };
    }

    // 2. 预先验证和加载文件树
    const treeResult = await FsService.listTree(folderPath);
    if (!treeResult.ok) {
      return { ok: false, error: treeResult.error };
    }

    // 3. 快照当前状态（用于回滚）
    const snapshot: WorkspaceSnapshot = {
      folders: useWorkspaceStore.getState().folders,
      rootFolders: useFileTreeStore.getState().rootFolders,
    };

    try {
      // 4. 构建新数据
      const newFolder: WorkspaceFolder = {
        path: folderPath,
        index: currentFolders.length
      };
      const newRoot: RootFolderNode = {
        workspacePath: folderPath,
        displayName: basename(folderPath),
        tree: treeResult.data,
      };

      // 5. 批量更新
      useWorkspaceStore.getState().addFolder(newFolder);
      useFileTreeStore.getState().addRootFolder(newRoot);

      // 6. 标记 isDirty
      useWorkspaceStore.getState().setDirty(true);

      // 7. 持久化状态
      await WorkspaceStatePersistence.saveCurrentState();

      return { ok: true };

    } catch (error) {
      // 8. 回滚
      useWorkspaceStore.setState({ folders: snapshot.folders });
      useFileTreeStore.setState({ rootFolders: snapshot.rootFolders });
      return { ok: false, error: String(error) };
    }
  },

  removeFolderFromWorkspace: async (folderPath: string): Promise<Result> => {
    // 1. 快照
    const snapshot: WorkspaceSnapshot = {
      folders: useWorkspaceStore.getState().folders,
      rootFolders: useFileTreeStore.getState().rootFolders,
    };

    try {
      // 2. 关闭该文件夹内已打开的文件
      const filesToClose = useWorkspaceStore.getState().openFiles
        .filter(p => p.startsWith(folderPath));

      for (const file of filesToClose) {
        await workspaceActions.closeFile(file);
      }

      // 3. 更新状态
      useWorkspaceStore.getState().removeFolder(folderPath);
      useFileTreeStore.getState().removeRootFolder(folderPath);
      useWorkspaceStore.getState().setDirty(true);

      // 4. 持久化
      await WorkspaceStatePersistence.saveCurrentState();

      return { ok: true };

    } catch (error) {
      // 5. 回滚
      useWorkspaceStore.setState({ folders: snapshot.folders });
      useFileTreeStore.setState({ rootFolders: snapshot.rootFolders });
      return { ok: false, error: String(error) };
    }
  },

  // ... 其他方法
};
```

```rust
// src-tauri/src/security.rs

use std::path::{Path, PathBuf};

/// 工作区路径验证器
pub struct WorkspaceAllowlist {
    roots: Vec<PathBuf>,
}

impl WorkspaceAllowlist {
    pub fn new(roots: Vec<String>) -> Self {
        Self {
            roots: roots.into_iter().map(PathBuf::from).collect(),
        }
    }

    /// 验证路径是否在工作区边界内
    ///
    /// # Arguments
    /// * `path` - 要验证的路径
    /// * `must_exist` - 路径是否必须已存在（新文件创建场景为 false）
    pub fn validate_path(&self, path: &str, must_exist: bool) -> Result<PathBuf, SecurityError> {
        let path_buf = PathBuf::from(path);

        // 路径穿越检查（显式）
        if path.contains("..") {
            return Err(SecurityError::PathTraversal);
        }

        // 已存在路径：验证完整路径
        if must_exist && path_buf.exists() {
            let canonical = path_buf.canonicalize()
                .map_err(|_| SecurityError::InvalidPath)?;
            return self.check_boundary(canonical);
        }

        // 新路径：验证父目录在工作区内
        let parent = path_buf.parent().ok_or(SecurityError::InvalidPath)?;
        if parent.exists() {
            let canonical_parent = parent.canonicalize()
                .map_err(|_| SecurityError::InvalidPath)?;
            self.check_boundary(canonical_parent)?;
        }

        // 返回原始路径（不 canonicalize，因为文件不存在）
        Ok(path_buf)
    }

    /// 检查路径是否在任何工作区根目录内
    fn check_boundary(&self, canonical: PathBuf) -> Result<PathBuf, SecurityError> {
        if !self.roots.iter().any(|root| canonical.starts_with(root)) {
            return Err(SecurityError::OutsideWorkspace);
        }
        Ok(canonical)
    }

    /// 添加工作区根目录
    pub fn add_root(&mut self, path: String) {
        if let Ok(canonical) = PathBuf::from(path).canonicalize() {
            // 验证新根目录的有效性
            if !self.roots.is_empty() {
                // 新根目录必须在现有根目录的边界内，或是独立的合法路径
                // 简化处理：允许添加任意合法路径作为新根
            }
            self.roots.push(canonical);
        }
    }

    /// 清除所有根目录（用于测试）
    pub fn clear_roots(&mut self) {
        self.roots.clear();
    }
}

#[derive(Debug)]
pub enum SecurityError {
    InvalidPath,
    OutsideWorkspace,
    PathTraversal,
}

impl std::fmt::Display for SecurityError {
    fn fmt(&self, f: &mut std::fmt::Formatter) -> std::fmt::Result {
        match self {
            SecurityError::InvalidPath => write!(f, "Invalid path"),
            SecurityError::OutsideWorkspace => write!(f, "Path outside workspace"),
            SecurityError::PathTraversal => write!(f, "Path traversal detected"),
        }
    }
}
```

```rust
// src-tauri/src/fs.rs

use std::thread;
use std::time::Duration;
use std::sync::Arc;
use crate::security::WorkspaceAllowlist;
use tauri::State;

/// 批量加载多个文件夹的树（并行实现 + 安全验证）
#[tauri::command]
pub fn list_tree_batch(
    paths: Vec<String>,
    allowlist: State<WorkspaceAllowlist>
) -> Result<Vec<FolderPathResult>, String> {
    // 输入验证：限制最大路径数量
    if paths.is_empty() {
        return Ok(vec![]);
    }

    if paths.len() > 20 {
        return Err("Too many paths (max 20)".to_string());
    }

    // 安全验证：所有路径必须在工作区内
    for path in &paths {
        allowlist.validate_path(path, true)
            .map_err(|e| format!("Security: {}", e))?;
    }

    // 受限线程池：最大 8 个并发线程
    let max_concurrent = std::cmp::min(paths.len(), 8);
    let (tx, rx) = std::sync::mpsc::channel();

    // 使用信号量限制并发
    let semaphore = Arc::new(tokio::sync::Semaphore::new(max_concurrent));

    let handles: Vec<_> = paths
        .into_iter()
        .map(|path| {
            let permit = semaphore.clone().acquire_owned().await.unwrap();
            let allowlist = allowlist.inner().clone();

            thread::spawn(move || {
                let result = list_tree(&path).map(|nodes| FolderPathResult {
                    path,
                    nodes,
                    error: None,
                }).unwrap_or_else(|e| FolderPathResult {
                    path: String::new(),
                    nodes: vec![],
                    error: Some(e),
                });
                drop(permit); // 释放信号量
                result
            })
        })
        .collect();

    // 收集结果
    handles
        .into_iter()
        .map(|h| h.join().unwrap_or_else(|_| FolderPathResult {
            path: String::new(),
            nodes: vec![],
            error: Some("Thread panic".to_string()),
        }))
        .collect()
}

/// 解析工作区文件
#[tauri::command]
pub fn parse_workspace_file(path: &str, allowlist: State<WorkspaceAllowlist>) -> Result<WorkspaceConfig, String> {
    // 安全验证
    allowlist.validate_path(path, true)
        .map_err(|e| format!("Security: {}", e))?;

    let content = fs::read_to_string(path).map_err(|e| e.to_string())?;
    let config: WorkspaceConfig = serde_json::from_str(&content)
        .map_err(|e| format!("Failed to parse workspace file: {}", e))?;
    Ok(config)
}

/// 保存工作区文件（使用 tempfile 确保原子性 + 安全验证）
#[tauri::command]
pub fn save_workspace_file(
    path: &str,
    config: WorkspaceConfig,
    allowlist: State<WorkspaceAllowlist>
) -> Result<(), String> {
    use std::io::Write;
    use tempfile::NamedTempFile;

    // 安全验证（新文件，must_exist=false）
    allowlist.validate_path(path, false)
        .map_err(|e| format!("Security: {}", e))?;

    let content = serde_json::to_string_pretty(&config)
        .map_err(|e| e.to_string())?;

    let target_path = Path::new(path);
    let parent = target_path.parent().ok_or("Invalid path")?;

    // 使用随机名称的临时文件
    let mut temp_file = NamedTempFile::new_in(parent)
        .map_err(|e| e.to_string())?;

    // Unix 系统设置文件权限（0o600，仅用户可读写）
    #[cfg(unix)]
    {
        use std::os::unix::fs::PermissionsExt;
        std::fs::set_permissions(temp_file.path(), std::fs::Permissions::from_mode(0o600))
            .map_err(|e| e.to_string())?;
    }

    temp_file.write_all(content.as_bytes())
        .map_err(|e| e.to_string())?;

    temp_file.persist(target_path)
        .map_err(|e| format!("Failed to persist file: {}", e))?;

    Ok(())
}

/// 解析相对路径为绝对路径（带安全验证）
#[tauri::command]
pub fn resolve_relative_path(
    base_path: &str,
    relative_path: &str,
    workspace_roots: Vec<String>,
    allowlist: State<WorkspaceAllowlist>
) -> Result<String, String> {
    // 检查路径穿越
    if relative_path.contains("..") {
        return Err("Path traversal not allowed".to_string());
    }

    let base = Path::new(base_path);
    let parent = base.parent().ok_or("Invalid base path")?;
    let resolved = parent.join(relative_path);

    // 规范化路径（不解析符号链接，更安全）
    let canonical = if resolved.exists() {
        resolved.canonicalize()
            .map_err(|e| format!("Cannot resolve path: {}", e))?
    } else {
        resolved
    };

    // 验证在工作区边界内
    let allowed = workspace_roots.iter().any(|root| {
        canonical.starts_with(PathBuf::from(root))
    });

    if !allowed {
        return Err("Resolved path outside workspace".to_string());
    }

    Ok(canonical.to_string_lossy().replace('\\', "/"))
}
```

```rust
// src-tauri/src/watcher.rs

use notify::{Watcher, RecursiveMode, watcher, Event};
use std::sync::mpsc::channel;
use std::time::Duration;
use std::sync::{Arc, Mutex};
use tauri::State;

/// 全局文件监听器状态
pub struct WatcherState {
    watcher: Option<Arc<Mutex<notify::RecommendedWatcher>>>,
    watched_paths: Vec<PathBuf>,
}

impl WatcherState {
    pub fn new() -> Self {
        Self {
            watcher: None,
            watched_paths: vec![],
        }
    }
}

/// 开始监听文件变化
#[tauri::command]
pub fn start_watching(
    app: tauri::AppHandle,
    paths: Vec<String>,
    state: State<Mutex<WatcherState>>,
) -> Result<(), String> {
    let mut guard = state.lock().unwrap();

    // 已存在则先停止旧监听
    if guard.watcher.is_some() {
        drop(guard);
        stop_watching(state.clone())?;
        guard = state.lock().unwrap();
    }

    let (tx, rx) = channel();

    // 创建监听器（300ms 防抖）
    let mut watcher = watcher(tx, Duration::from_millis(300))
        .map_err(|e| e.to_string())?;

    // 添加监听路径
    for path in &paths {
        watcher.watch(Path::new(path), RecursiveMode::Recursive)
            .map_err(|e| e.to_string())?;
    }

    // 将 watcher 存储到全局状态（使用 Arc 包装，避免被 drop）
    let watcher_arc = Arc::new(Mutex::new(watcher));

    // 在后台线程中转发事件到前端
    let watcher_clone = watcher_arc.clone();
    let app_clone = app.clone();
    std::thread::spawn(move || {
        // 持有 watcher_arc 的引用，防止 watcher 被 drop
        let _watcher_ref = watcher_clone;

        while let Ok(event) = rx.recv() {
            // 发送事件到前端
            let _ = app_clone.emit("writer://file-change", FileChangeEvent {
                kind: format!("{:?}", event.kind),
                paths: event.paths.iter().map(|p| p.to_string_lossy().to_string()).collect(),
            });
        }
    });

    // 更新全局状态
    guard.watcher = Some(watcher_arc);
    guard.watched_paths = paths.into_iter().map(PathBuf::from).collect();

    Ok(())
}

/// 停止监听
#[tauri::command]
pub fn stop_watching(
    state: State<Mutex<WatcherState>>
) -> Result<(), String> {
    let mut guard = state.lock().unwrap();

    if let Some(watcher_arc) = guard.watcher.take() {
        // 显式 drop watcher，停止监听
        drop(watcher_arc);
    }

    guard.watched_paths.clear();

    Ok(())
}

/// 更新监听路径
#[tauri::command]
pub fn update_watch_paths(
    app: tauri::AppHandle,
    new_paths: Vec<String>,
    state: State<Mutex<WatcherState>>,
) -> Result<(), String> {
    // 停止旧监听
    stop_watching(state.clone())?;
    // 启动新监听
    start_watching(app, new_paths, state)
}

// 前端事件结构
#[derive(Debug, serde::Serialize, Clone)]
pub struct FileChangeEvent {
    pub kind: String,
    pub paths: Vec<String>,
}
```

### 5.3 FileWatcherService（修复防抖逻辑）

```typescript
// src/services/filewatcher/FileWatcherService.ts

import { listen } from '@tauri-apps/api/event';
import { invoke } from '@tauri-apps/api/core';

export interface FileChangeEvent {
  type: 'add' | 'change' | 'unlink' | 'unlinkDir' | 'rename';
  path: string;
  oldPath?: string;
}

export const FileWatcherService = {
  isWatching: false,
  currentPaths: [] as string[],
  eventListeners: Array<(event: FileChangeEvent) => void>(),

  // 修复：防抖定时器提升为成员变量
  private debounceTimers: Map<string, NodeJS.Timeout> = new Map(),

  // 开始监听工作区
  async startWatching(
    workspaceFolders: string[],
    onChange: (event: FileChangeEvent) => void
  ): Promise<void> {
    if (this.isWatching) {
      await this.stopWatching();
    }

    // 注册事件监听器
    this.eventListeners.push(onChange);

    // 监听后端推送的事件
    const unlisten = await listen('writer://file-change', (event: any) => {
      this.handleBackendEvent(event.payload);
    });

    // 调用后端启动监听
    await invoke('start_watching', { paths: workspaceFolders });

    this.isWatching = true;
    this.currentPaths = workspaceFolders;

    // 保存 unlisten 函数以便清理
    (this as any).unlisten = unlisten;
  },

  // 修复：处理后端事件（防抖逻辑正确）
  private handleBackendEvent(payload: any): void {
    // 映射 notify 事件到统一格式
    const events = this.mapEvent(payload);

    // 防抖处理（300ms）
    events.forEach(event => {
      if (this.debounceTimers.has(event.path)) {
        clearTimeout(this.debounceTimers.get(event.path)!);
      }

      const timer = setTimeout(() => {
        this.eventListeners.forEach(cb => cb(event));
        this.debounceTimers.delete(event.path);
      }, 300);

      this.debounceTimers.set(event.path, timer);
    });
  },

  // 映射事件类型
  private mapEvent(payload: any): FileChangeEvent[] {
    return payload.paths.map((path: string) => ({
      type: this.mapKind(payload.kind),
      path,
    }));
  },

  private mapKind(kind: string): FileChangeEvent['type'] {
    if (kind.includes('Create')) return 'add';
    if (kind.includes('Modify')) return 'change';
    if (kind.includes('Remove')) return 'unlink';
    if (kind.includes('Rename')) return 'rename';
    return 'change';
  },

  // 停止监听
  async stopWatching(): Promise<void> {
    if (!this.isWatching) return;

    await invoke('stop_watching');
    if ((this as any).unlisten) {
      (this as any).unlisten();
    }

    this.isWatching = false;
    this.currentPaths = [];
    this.eventListeners = [];

    // 清理所有定时器
    this.debounceTimers.forEach(timer => clearTimeout(timer));
    this.debounceTimers.clear();
  },

  // 重新监听（工作区变更后）
  async restartWatching(
    newWorkspaceFolders: string[],
    onChange: (event: FileChangeEvent) => void
  ): Promise<void> {
    await this.stopWatching();
    await this.startWatching(newWorkspaceFolders, onChange);
  },
};
```

## 10. 风险与缓解

### 10.1 技术风险

| 风险                     | 影响   | 可能性 | 缓解措施                   |
| ------------------------ | ------ | ------ | -------------------------- |
| 状态不一致               | 高     | 中     | 快照 - 回滚机制 + 严格测试 |
| 多根文件树渲染性能       | 中     | 中     | 虚拟滚动 + 懒加载          |
| 拖拽跨平台兼容性         | 中     | 中     | 降级到对话框               |
| 后端命令性能             | 中     | 低     | 并行化 + 缓存              |
| 外部文件监听性能         | 中     | 高     | 防抖 + 排除模式            |
| Sidebar 改造超期         | 中     | 中     | 拆分组件，增加工时估算     |
| **watcher 生命周期泄漏** | **高** | **低** | **全局状态管理（已修复）** |
| **前端防抖失效**         | **中** | **低** | **成员变量存储（已修复）** |

### 10.2 缓解措施详情

**状态不一致**:

- 使用 `workspaceActions` 统一协调
- 添加快照 - 回滚机制
- 所有多 store 更新在 try-catch 中完成
- 添加状态验证断言

**渲染性能**:

- 实现虚拟滚动（`react-window`）
- 懒加载子树（按需加载）
- 限制单次渲染节点数（<1000）

**外部文件监听性能**:

- 使用 Rust `notify` crate 后端监听
- 配置排除模式（`node_modules`、`.git` 等）
- 300ms 防抖避免频繁事件触发
- 大文件夹（>1000 文件）显示性能警告

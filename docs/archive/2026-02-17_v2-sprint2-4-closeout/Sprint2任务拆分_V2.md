# Sprint2 任务拆分（V2 执行）

## 目标

完成快速文件搜索与查找/替换能力，保证编辑链路与自动保存链路稳定。

## 任务列表

- [ ] S2-V2-01 快速文件搜索 MVP（入口、输入过滤、结果列表打开文件）
- [ ] S2-V2-02 查找/替换 MVP（查找、上一处/下一处、单次替换）
- [ ] S2-V2-03 搜索结果与文件切换一致性（切换文件不丢失未保存状态）
- [ ] S2-V2-04 替换链路与撤销/重做兼容性验证
- [ ] S2-V2-05 回归门禁（表格、顶部菜单、自动保存、文件树）

## DoD

1. 工作区内可通过统一入口快速定位并打开目标文件。
2. 当前文档查找/替换结果定位准确，单次替换行为可预期。
3. 文件切换后，未保存编辑状态不丢失。
4. 替换后撤销/重做链路正常。
5. Sprint2 任务与 QA 在追溯矩阵中有明确映射与证据路径。

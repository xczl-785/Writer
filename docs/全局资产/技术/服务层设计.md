## 二、服务职责清单

| 服务                | 职责                                | 后端通信          | 依赖                          |
| ------------------- | ----------------------------------- | ----------------- | ----------------------------- |
| **FsService**       | 文件/目录 CRUD、Git 状态、编码检测  | `invoke`          | Tauri                         |
| **MarkdownService** | Markdown ⇔ EditorJSON 双向转换      | 无                | 无                            |

| **FsSafety**        | 文件操作前脏文件 flush              | AutosaveService   | AutosaveService, ErrorService |
| **TauriWarmup**     | Tauri 运行时桥接预热                | `invoke('greet')` | Tauri                         |

## 三、与 Tauri 后端通信

### 3.1 通信模式

所有后端通信集中在 `FsService.ts` 中，使用 `invoke` 调用：

```typescript
import { invoke } from '@tauri-apps/api/core';

// 读取文件
const content = await invoke('read_file', { path });

// 写入文件
await invoke('write_file_atomic', { path, content });

// 列出目录树
const tree = await invoke('list_tree', { path });
```

### 3.2 Tauri 命令清单

| 命令                     | 功能                        |
| ------------------------ | --------------------------- |
| `list_tree`              | 递归列出目录树              |
| `read_file`              | 读取文件内容                |
| `write_file_atomic`      | 原子写入（临时文件+重命名） |
| `create_file`            | 创建新文件                  |
| `create_dir`             | 创建新目录                  |
| `rename_node`            | 重命名/移动文件或目录       |
| `delete_node`            | 删除文件或目录              |
| `reveal_in_file_manager` | 在文件管理器中显示          |
| `save_image`             | 保存图片（二进制）          |
| `check_exists`           | 检查路径是否存在            |
| `get_git_sync_status`    | 获取 Git 同步状态           |
| `detect_file_encoding`   | 检测文件编码                |

## 五、关键服务详解

### 5.1 FsService

文件系统操作的统一入口，封装所有 Tauri `invoke` 调用。

**设计原则**：

- 单一入口：所有文件操作都通过 FsService
- 错误处理：统一返回类型，便于上层处理
- 类型安全：完整的 TypeScript 类型定义

### 5.2 AutosaveService

防抖自动保存调度器。

**核心机制**：

- 防抖延迟（默认配置）
- 待保存队列管理
- 重试机制
- 手动 flush 接口

### 5.3 MarkdownService

Markdown 与 TipTap JSON 双向转换。

**API**：

```typescript
// Markdown → TipTap JSON
const json = await MarkdownService.parse(markdown);

// TipTap JSON → Markdown
const markdown = await MarkdownService.serialize(json);
```

### 5.4 ImageResolver

解析图片相对/绝对路径为 Tauri asset URL。

**使用场景**：

- 编辑器中显示本地图片
- 导出时处理图片路径

## 六、设计模式

### 6.1 工具服务单例

所有服务使用对象字面量实现：

```typescript
export const FsService = {
  async readFile(path: string): Promise<string> {
    return invoke('read_file', { path });
  },
  // ...
};
```

### 6.2 依赖注入缺失

当前服务间通过直接 import 引用，测试时需要 mock。

**潜在改进**：

```typescript
// 可考虑引入依赖注入
type FsServiceInterface = {
  readFile: (path: string) => Promise<string>;
  // ...
};

export function createServices(fs: FsServiceInterface) {
  return {
    autosave: createAutosaveService(fs),
    // ...
  };
}
```

## 五、核心功能实现

### 5.1 工具栏

- 命令定义在 `constants.ts` 的 `TOOLBAR_COMMANDS` 数组
- `useToolbarCommands` hook 封装命令执行逻辑
- 按功能分组：Inline、Headings、Blocks

### 5.2 斜杠菜单

- 输入 `/` 触发（支持全角 `／`）
- 支持搜索过滤、键盘导航
- 命令分组：Basic Blocks、Advanced Components
- 实现：`ui/menus/useSlashMenu.ts`、`ui/menus/SlashMenuView.tsx`

### 5.3 浮动菜单

- 选中文字时出现
- 80ms 防抖延迟
- 包含：B、I、S、</>、Link、Highlight 按钮
- 实现：`ui/menus/BubbleMenu.tsx`

### 5.4 查找替换

- 快捷键：Cmd+F（查找）、Cmd+H（替换）
- 支持单个/全部替换、上下导航
- 实现：`hooks/useFindReplace.ts`

### 5.5 滚动协调器

- 统一管理编辑器滚动事件
- 支持多个滚动源：打字机模式、大纲导航、Slash 菜单、查找替换
- 提供 `ScrollCoordinator` 协调器接口
- 实现：`domain/scroll/`
- 详细文档：[滚动协调器设计](./滚动协调器设计.md)

### 5.6 打字机模式（Typewriter V2.1）

- 核心实现基于事件驱动状态机（`Free / Locked`）。
- 领域分层：
  - 事件定义：`domain/typewriter/events.ts`
  - 守卫规则：`domain/typewriter/guards.ts`
  - 状态归约：`domain/typewriter/reducers.ts`
  - 状态机基础：`domain/typewriter/typewriterStateMachine.ts`
- UI 接入：`hooks/useTypewriterAnchor.ts` 负责事件采集、状态分发、补偿执行。
- 非输入跳转（Find/Outline/程序化选区）通过 `TYPEWRITER_FORCE_FREE_EVENT` 回退 `Free`。
- 详细文档：[打字机模式设计](./打字机模式设计.md)

## 六、编辑器与状态同步

### 6.1 内容更新流程

```
用户输入 → TipTap Editor.onUpdate
    │
    ▼
MarkdownService.serialize(editor.getJSON())
    │
    ▼
updateFileContent(path, markdown)
    │
    ▼
setDirty(path, true)
    │
    ▼
AutosaveService.schedule(path, markdown)
```

### 6.2 文件加载流程

```
activeFile 变化 → useEffect 触发
    │
    ▼
MarkdownService.parse(content)
    │
    ▼
editor.commands.setContent(json)
```

### 6.3 顶部偏移变量策略（Find 面板）

- 当前 Find 面板保持**文档流占位**（不悬浮覆盖正文）。
- 编辑器引入运行时变量 `--editor-content-offset-top`，用于表达“顶部遮挡差值”。
- 当前布局下该变量通常为 `0px`，不会产生双重偏移；若未来 Find 改为 overlay/sticky，可直接复用该变量而无需重写 Typewriter 锚点逻辑。
- `ui/components/EditorShell.tsx` 负责测量并写入变量（`ResizeObserver + useLayoutEffect`）；`hooks/useTypewriterAnchor.ts` 消费变量参与有效视口计算。
- 本轮仅接入 Find 来源，暂不做多源顶部占位聚合（后续可扩展）。

## 七、扩展开发指南

### 7.1 创建新扩展

```typescript
// extensions/myExtension.ts
import { Extension } from '@tiptap/core';

export const MyExtension = Extension.create({
  name: 'my-extension',
  addKeyboardShortcuts() {
    return {
      'Mod-m': () => {
        // 执行操作
        return true;
      },
    };
  },
});
```

### 7.2 注册扩展

在 `core/EditorImpl.tsx` 的 `extensions` 数组中添加：

```typescript
const extensions = useMemo(
  () => [
    // ... 其他扩展
    MyExtension,
  ],
  [],
);
```

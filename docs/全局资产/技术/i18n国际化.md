# i18n 国际化

**更新日期**: 2026-03-02

---

## 一、架构概述

Writer 采用轻量级自定义 i18n 方案，支持 `zh-CN` 和 `en-US` 两种语言，根据用户系统语言自动切换。

```
src/i18n/
├── index.ts       # 核心模块：getLocale, setLocale, t 函数
└── messages.ts    # 翻译字典：所有文案的 key-value 映射
```

---

## 二、核心 API

### 2.1 导入方式

```typescript
import { t, getLocale, setLocale } from '../../i18n';
```

### 2.2 翻译函数 `t(key: string): string`

```typescript
// 基础用法
t('menu.file'); // → '文件' / 'File'
t('sidebar.newFile'); // → '新建文件' / 'New File'

// 带插值的文案（手动替换）
t('confirm.deleteFolder').replace('{name}', fileName);
```

### 2.3 语言获取与切换

```typescript
// 获取当前语言
const locale = getLocale(); // 'zh-CN' | 'en-US'

// 切换语言（运行时切换）
setLocale('en-US');
```

### 2.4 语言检测逻辑

```typescript
// 默认跟随系统语言
const defaultLocale = navigator.language.startsWith('zh') ? 'zh-CN' : 'en-US';
```

## 三、翻译键命名规范

### 3.1 命名空间划分

| 命名空间        | 覆盖模块                 | 示例 key              |
| --------------- | ------------------------ | --------------------- |
| `menu.*`        | 顶部菜单栏               | `menu.file.save`      |
| `sidebar.*`     | 文件树侧边栏             | `sidebar.newFile`     |
| `status.*`      | 状态栏、保存状态         | `status.saved`        |
| `editor.*`      | 编辑器右键菜单、表格操作 | `editor.insertTable`  |
| `slash.*`       | 斜杠菜单命令             | `slash.heading1`      |
| `dialog.*`      | 查找替换弹窗             | `dialog.find`         |
| `table.*`       | 插入表格弹窗             | `table.rows`          |
| `confirm.*`     | 确认对话框               | `confirm.deleteTitle` |
| `error.*`       | 错误消息                 | `error.saveFailed`    |
| `close.*`       | 关闭工作流               | `close.saveFailed`    |
| `contextMenu.*` | 右键菜单                 | `contextMenu.rename`  |

### 3.2 命名风格

- 使用 **camelCase**：`menu.file.save`
- 语义清晰，避免缩写
- 模块前缀 + 功能描述：`sidebar.newFolderPlaceholder`

## 四、添加新文案流程

### 4.1 步骤

1. 在 `src/i18n/messages.ts` 中添加翻译键
2. 在组件中导入 `t` 函数
3. 替换硬编码字符串

### 4.2 示例

**Step 1：添加翻译键**

```typescript
// src/i18n/messages.ts
export const MESSAGES: Record<AppLocale, Messages> = {
  'zh-CN': {
    // ...existing keys
    'myModule.myText': '中文文案',
  },
  'en-US': {
    // ...existing keys
    'myModule.myText': 'English text',
  },
};
```

**Step 2：在组件中使用**

```typescript
import { t } from '../../i18n';

function MyComponent() {
  return <button>{t('myModule.myText')}</button>;
}
```

```typescript

// 如需 ICU 支持，可引入 intl-messageformat 库
```

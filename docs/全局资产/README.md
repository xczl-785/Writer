## 使用规则

- 与单一版本强绑定的文档不得进入本目录。
- 本目录文档发生变更时，必须同步更新：
  - `docs/全局资产/治理/决策日志.md`
  - 对应 `docs/archive/**` 评审记录
- 阶段性执行文档应留在 `docs/current/`，完成后归档到 `docs/archive/`。

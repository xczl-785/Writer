V5 状态反馈流转演示 (Status Indicator)

根据 V5 规范，通知弹窗被大面积削减。所有文档的状态变化收敛于底部栏左侧的这枚指示器。
它将经历 **编辑(黄) -> 保存中(蓝/呼吸) -> 已保存(绿) -> 隐匿(低透明度)** 的四态流转。

状态 1: 未保存 (用户发生编辑动作 1s 内)

Unsaved

状态 2: 保存中 (触发自动保存或 Cmd+S 时)

Saving...

状态 3: 已保存 (保存成功后，短暂高亮)

Saved

状态 4: 稳态隐匿 (保持已保存状态 > 5 秒)

Saved

异常分支: 无法保存 (例如遭遇无权限被 Tauri 拦截)

Save Failed
系统级异常

Writer 无法覆写当前文件。可能是该文件被其他程序锁定或您没有写入权限。

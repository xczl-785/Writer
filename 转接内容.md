## PM 从业习惯：迭代式推进

```
第1步：填充确定的 → 形成可评审版本
第2步：评审 → 对齐认知
第3步：确定阻塞项 → 并行调研
第4步：迭代补充 → 逐步完善
```

**核心理念**：

- 不追求一次性完美
- 先产出可评审版本
- 评审反馈比完美规划更重要
